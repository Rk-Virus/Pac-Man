=================================================
                PACMAN GAME
=================================================

Thank you for downloading PacMan! This classic arcade game features 5 levels
of increasing difficulty with multiple ghosts to avoid while collecting all
the food pellets.

-------------------------------------------------
HOW TO RUN THE GAME
-------------------------------------------------

Option 1: Use the PacMan_Launcher.bat (Recommended)
   - Double-click on PacMan_Launcher.bat
   - Select option 1 to play the game directly (requires Java)
   - Select option 2 to create a standalone Windows executable (requires JDK 17+)

Option 2: Run directly
   - Double-click on play_pacman.bat to start the game (requires Java)

Option 3: Create and use executable
   - Double-click on create_exe.bat to create a standalone executable (requires JDK 17+)
   - Once created, you can run PacMan.exe directly without Java installed

-------------------------------------------------
SYSTEM REQUIREMENTS
-------------------------------------------------

To play using the JAR file (Options 1 or 2):
   - Java Runtime Environment (JRE) 8 or higher
   - Windows operating system
   - Minimum screen resolution: 800x600

To create the executable (Option 3):
   - Java Development Kit (JDK) 17 or higher
   - Windows operating system

To run the executable:
   - Windows operating system
   - No Java installation required

-------------------------------------------------
GAME CONTROLS
-------------------------------------------------

- Arrow Keys: Move PacMan
- Close window to exit game

-------------------------------------------------
DISTRIBUTION
-------------------------------------------------

If you want to share this game with others:

1. For users with Java installed:
   - Share the entire folder or create a ZIP with these files:
     * build/PacManGame.jar
     * play_pacman.bat

2. For users without Java:
   - Create the executable using Option 3 above
   - Share only the generated PacMan.exe file

-------------------------------------------------
TROUBLESHOOTING
-------------------------------------------------

If the game doesn't start:

1. Make sure Java is installed (for JAR version)
   - Download from: https://www.java.com/download/

2. If creating the executable fails:
   - Ensure you have JDK 17+ installed
   - Download from: https://www.oracle.com/java/technologies/downloads/
   - Make sure JDK is added to your PATH environment variable
